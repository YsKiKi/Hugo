### 暂停 Windows 系统更新
按下`win+x`并选择`终端(管理员)`，输入后回车：
```
reg add "HKLM\SOFTWARE\Microsoft\WindowsUpdate\UX\Settings" /v FlightSettingsMaxPauseDays /t REG_DWORD /d 65535 /f
```
打开系统更新，可将更新时间推迟到很久之后，选择下拉框时可能卡顿。

### 将右键菜单改为旧版本(Windows10风格)
实际上是禁用 Windows11新的右键菜单组件。  
按下`win+x`并选择`终端(管理员)`，输入后回车：
```
reg add "HKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}\InprocServer32" /ve /d "" /f
```
此项更改成功后需重启文件资源管理器(explorer.exe)才可生效。
